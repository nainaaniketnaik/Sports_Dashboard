# Sports_Dashboard
# 🏆 Sports Analysis Dashboard | Power BI

## 📌 Overview
This Power BI dashboard provides comprehensive insights into sports performance across multiple sports categories. It helps analyze wins, losses, goals, fouls, player performance, analyst contributions, and monthly trends through interactive visualizations.

## 📊 Dashboard Highlights

### Key Metrics
- Total Matches Played: 1,000
- Total Wins: 231
- Total Losses: 269
- Total Goals: 970
- Total Fouls: 5,000+
- Total Draws: 132
- Total Assists: 582
- Total Passes: 141K
- Distance Covered: 4.29K km

## 📈 Features
- Sport-wise Win vs Loss Analysis
- Monthly Match Performance Tracking
- Goals and Assists Analysis
- Player Rating & Distance Covered Analysis
- Venue-wise Performance Insights
- Yellow Card and Red Card Monitoring
- Interactive Filters and Drill-through Reports

## 🛠️ Tools Used
- Power BI Desktop

- Data Modeling

## 🔍 Key Insights
- Softball recorded the highest number of wins.
- Sports performance varies significantly across categories.
- Monthly trends reveal fluctuations in match activity.
- Analyst performance can be measured through goals and assists.
- Venue analysis helps identify high-performing locations.
